# Screenshots

这个文件夹包含Stocks-Lab项目的功能截图和演示图片。

## 文件说明

- `account-detail.png` - 账户详情页面
- `trade-share.png` - 交易分享功能
- `completed-trades.png` - 撮合完成的交易列表
- `trade-note.png` - 交易笔记导出
- `position-share.png` - 持仓分享功能

## 上传截图

请将项目截图放入此文件夹，截图文件名请使用英文命名，便于版本控制。

- 分辨率：1920x1080 或更高
