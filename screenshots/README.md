# Screenshots

这个文件夹包含Stocks-Lab项目的功能截图和演示图片。

## 已上传截图

- `CHs.png` - 中文界面截图 (115KB)
  - 展示中文界面的完整功能
  - 包含撮合交易列表、交易分享等核心功能
  
- `Eng.png` - 英文界面截图 (101KB)
  - 展示英文界面的完整功能
  - 国际化支持的视觉效果

## 建议补充截图

为了更完整地展示项目功能，建议添加以下截图：

- `account-detail.png` - 账户详情页面，显示撮合交易列表
- `trade-share.png` - 交易分享页面，显示撮合交易和生成按钮
- `completed-trade-detail.png` - 撮合交易详情模态框
- `trade-share-image.png` - 生成的交易分享图片示例
- `trade-note.png` - 导出的交易笔记示例
- `position-share.png` - 持仓分享功能界面

## 截图要求

- **分辨率**：1920x1080 或更高
- **格式**：PNG（推荐）或 JPG
- **内容**：展示核心功能界面，避免敏感信息
- **大小**：单个文件不超过 2MB
- **命名**：使用英文命名，便于版本控制

## 上传方式

1. 将截图文件复制到此文件夹
2. 运行 `git add screenshots/` 添加文件
3. 运行 `git commit -m "feat: add screenshots"` 提交
4. 运行 `git push origin main` 推送到GitHub
